1. Project Description

Taxi Fare Analysis Dashboard – Power BI

Developed an interactive Power BI dashboard to analyze taxi trip data, fare distribution, trip distances, passenger counts, and payment methods. The dashboard provides insights into travel patterns, revenue trends, peak demand periods, and key performance indicators (KPIs), enabling data-driven decision-making for taxi services.

2. Tech Stack
Power BI Desktop – Dashboard development and visualization
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated columns and measures
Microsoft Excel / CSV – Data source
Data Modeling – Relationships and data organization
3. Project Highlights
Cleaned and transformed raw taxi trip data using Power Query.
Created interactive dashboards with slicers and filters.
Analyzed total fare, trip distance, passenger count, and payment methods.
Built KPIs for total revenue, average fare, average trip distance, and total trips.
Visualized trends using bar charts, line charts, maps, pie charts, and cards.
Identified peak travel hours and high-demand locations.
Used DAX measures to calculate business metrics and improve dashboard insights.
4. Data Source
Dataset: Taxi Fare Records (Yellow Taxi Dataset) or Taxi Fare Dataset (CSV)
Format: CSV / Excel
